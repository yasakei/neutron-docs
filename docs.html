Neutron Language Reference

This document provides a comprehensive reference for the Neutron programming language, covering syntax, data types, operators, control flow, functions, and other features.

Getting Started

Welcome to Neutron! This guide will help you get up and running with the Neutron programming language in no time.

What is Neutron?

Neutron is a simple, modern, and lightweight interpreted programming language designed for ease of use and learning. It combines familiar C-like syntax with dynamic typing and object-oriented features.

Key Features

  • Dynamic typing - no need to declare variable types
  • C-like syntax familiar to many developers
  • Object-oriented programming with classes and methods
  • Rich standard library for common tasks
  • Fast C++ implementation for optimal performance
  • Interactive REPL for quick experimentation

Installation

Prerequisites

To build and run Neutron, you will need:

  • A C++ compiler that supports C++17 (GCC 7+, Clang 5+, or MSVC 2017+)
  • Make utility
  • Git (for cloning the repository)

Building from Source

# Clone the repository
git clone https://github.com/yasakei/neutron.git
cd neutron

# Build the interpreter
make

# Verify the installation
./neutron --version

Running Your First Program

Create a file called hello.nt with the following content: 

print("Hello, Neutron!");

Run the program: 

./neutron hello.nt

Interactive REPL

Start the interactive REPL (Read-Eval-Print Loop) for experimentation: 

./neutron

Syntax

Comments

Comments in Neutron start with //. Everything from // to the end of the line is ignored. 

// This is a comment
var x = 10; // This is also a comment

Data Types

Neutron is dynamically typed and has the following built-in data types:

Numbers

64-bit floating-point values

var x = 10;      // Integer
var y = 3.14;    // Float

Booleans

True or false values

var isActive = true;
var isAdmin = false;

Strings

Sequences of characters

var name = "Neutron";
var message = 'Hello!';

Nil

Absence of a value

var empty = nil;

Operators

Neutron supports a variety of operators for arithmetic, comparison, and logical operations.

Arithmetic Operators

Basic Operations

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
var x = 10 + 5;  // 15
var y = 20 - 10; // 10
var z = 5 * 2;   // 10
var w = 10 / 2;  // 5

Comparison Operators

Equality & Comparison

  • == Equal
  • != Not equal
  • < Less than
  • <= Less than or equal
  • > Greater than
  • >= Greater than or equal
print(10 == 10); // true
print(10 != 5);  // true
print(5 < 10);   // true
print(15 >= 10); // true

Logical Operators

Boolean Logic

  • and Logical AND
  • or Logical OR
  • ! Logical NOT
print(true and false); // false
print(true or false);  // true
print(!true);          // false

var a = 5;
var b = 10;
print(a > 0 and b < 20); // true

Variables

Variables in Neutron are declared using the var keyword. They are dynamically typed and can hold any value. 

var name = "Neutron";
var version = 1.0;
var isReady = true;

// Variables can be reassigned
name = "Neutron Language";
version = 2.0;

Functions

Functions are declared using the fun keyword. They can take parameters and return values. 

// Function declaration
fun greet(name) {
    return "Hello, " + name + "!";
}

// Function call
var message = greet("World");
print(message); // Output: Hello, World!

// Function with multiple parameters
fun add(a, b) {
    return a + b;
}

var sum = add(5, 3); // sum = 8

Classes

Neutron supports object-oriented programming with classes. Classes can have properties and methods, and support the this keyword. 

class Person {
    var name;
    var age;

    fun setName(name) {
        this.name = name;
    }

    fun setAge(age) {
        this.age = age;
    }

    fun introduce() {
        print("Hi, I'm " + this.name + " and I'm " + this.age + " years old.");
    }
}

// Create an instance
var person = Person();
person.setName("Alice");
person.setAge(30);
person.introduce(); // Output: Hi, I'm Alice and I'm 30 years old.

Control Flow

If Statements

var age = 18;

if (age >= 18) {
    print("You are an adult.");
} else {
    print("You are a minor.");
}

// Multiple conditions
var score = 85;
if (score >= 90) {
    print("A grade");
} else if (score >= 80) {
    print("B grade");
} else {
    print("C grade or below");
}

Loops

// While loop
var i = 0;
while (i < 5) {
    print(i);
    i = i + 1;
}

// For loop
for (var j = 0; j < 3; j = j + 1) {
    print("Iteration: " + j);
}

Modules

Neutron supports modular programming through import statements. The standard library includes modules for math, HTTP, JSON, and more. 

// Import math module
import "math";

var result = math.sqrt(16); // result = 4
var pi = math.PI; // pi = 3.14159...

// Import JSON module
import "json";

var data = json.parse('{"name": "Neutron", "version": 1.0}');
print(data.name); // Output: Neutron

Built-in Functions

Neutron provides several built-in functions for common operations:

print(value)

Outputs a value to the console

print("Hello, World!");
print(42);
print(true);

len(string)

Returns the length of a string

var length = len("Neutron"); // length = 7

type(value)

Returns the type of a value as a string

print(type(42));     // "number"
print(type("hi"));   // "string"
print(type(true));   // "boolean"

Standard Library

Neutron comes with a comprehensive standard library that provides modules for common programming tasks.

Math Module

The math module provides mathematical functions and constants. 

import "math";

// Mathematical functions
var result = math.sqrt(16);        // 4
var power = math.pow(2, 3);        // 8
var absolute = math.abs(-5);       // 5

// Trigonometric functions
var sine = math.sin(math.PI / 2);  // 1
var cosine = math.cos(0);          // 1
var tangent = math.tan(math.PI / 4); // 1

// Logarithmic functions
var natural_log = math.log(math.E); // 1
var exponential = math.exp(1);      // 2.718...

// Rounding functions
var ceiling = math.ceil(4.1);       // 5
var floor = math.floor(4.9);        // 4
var rounded = math.round(4.5);      // 5

JSON Module

The JSON module provides functions for parsing and generating JSON data. 

import "json";

// Parse JSON string to object
var jsonString = '{"name": "Neutron", "version": 1.0, "active": true}';
var data = json.parse(jsonString);

print(data.name);    // "Neutron"
print(data.version); // 1.0
print(data.active);  // true

// Convert object to JSON string
var obj = {"language": "Neutron", "year": 2024};
var jsonOutput = json.stringify(obj);
print(jsonOutput);   // {"language":"Neutron","year":2024}

HTTP Module

The HTTP module provides functions for making HTTP requests. 

import "http";

// Make a GET request
var response = http.get("https://api.example.com/data");
print(response.status);  // 200
print(response.body);    // Response content

// Make a POST request with data
var postData = {"key": "value"};
var postResponse = http.post("https://api.example.com/submit", postData);
print(postResponse.status);

Convert Module

The convert module provides data conversion utilities. 

import "convert";

// String conversions
var num = convert.toNumber("42");    // 42
var str = convert.toString(42);      // "42"

// Binary conversions
var binary = convert.toBinary(10);   // "1010"
var decimal = convert.fromBinary("1010"); // 10

// Character conversions
var ascii = convert.charToAscii("A"); // 65
var char = convert.asciiToChar(65);   // "A"

System Module

The sys module provides system-level operations and file system access. 

import "sys";

// User input
var name = sys.input("Enter your name: ");
var age = sys.input("Enter your age: ");

// File operations
sys.write("data.txt", "Hello, Neutron!");
var content = sys.read("data.txt");
print(content); // "Hello, Neutron!"

// File system operations
var exists = sys.exists("data.txt");  // true
sys.mkdir("new_directory");
sys.cp("data.txt", "backup.txt");

// System information
var currentDir = sys.cwd();
print("Current directory: " + currentDir);

var envVar = sys.env("PATH");
print("PATH: " + envVar);

// Execute system commands
var result = sys.exec("ls -la");
print(result.output);
print("Exit code: " + result.code);

Complete API Reference

This section provides a complete reference of all available functions, methods, and modules in Neutron.

Built-in Functions

print(value)

Outputs a value to the console followed by a newline.

Parameters: value (any) - The value to print

Returns: nil

print("Hello, World!");
print(42);
print(true);

len(value)

Returns the length of a string.

Parameters: value (string) - The string to measure

Returns: number - The length of the string

var length = len("Neutron"); // 7
var empty = len("");         // 0

type(value)

Returns the type of a value as a string.

Parameters: value (any) - The value to check

Returns: string - The type name

print(type(42));        // "number"
print(type("hello"));   // "string"
print(type(true));      // "boolean"
print(type(nil));       // "nil"
