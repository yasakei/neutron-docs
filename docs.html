Neutron Language Reference

This document provides a comprehensive reference for the Neutron programming language, covering syntax, data types, operators, control flow, functions, and other features.

Getting Started

Welcome to Neutron! This guide will help you get up and running with the Neutron programming language in no time.

What is Neutron?

Neutron is a simple, modern, and lightweight interpreted programming language designed for ease of use and learning. It combines familiar C-like syntax with dynamic typing and object-oriented features.

Key Features

  • Dynamic typing - no need to declare variable types
  • C-like syntax familiar to many developers
  • Object-oriented programming with classes and methods
  • Rich standard library for common tasks
  • Fast C++ implementation for optimal performance
  • Interactive REPL for quick experimentation

Installation

Prerequisites

To build and run Neutron, you will need:

  • A C++ compiler that supports C++17 (GCC 7+, Clang 5+, or MSVC 2017+)
  • Make utility
  • Git (for cloning the repository)

Building from Source

# Clone the repository
git clone https://github.com/yasakei/neutron.git
cd neutron

# Build the interpreter
make

# Verify the installation
./neutron --version

Running Your First Program

Create a file called hello.nt with the following content: 

say("Hello, Neutron!");

Run the program: 

./neutron hello.nt

Interactive REPL

Start the interactive REPL (Read-Eval-Print Loop) for experimentation: 

./neutron

Syntax

Comments

Comments in Neutron start with //. Everything from // to the end of the line is ignored. 

// This is a comment
var x = 10; // This is also a comment

Data Types

Neutron is dynamically typed and has the following built-in data types:

Numbers

64-bit floating-point values

var x = 10;      // Integer
var y = 3.14;    // Float

Booleans

True or false values

var isActive = true;
var isAdmin = false;

Strings

Sequences of characters

var name = "Neutron";
var message = 'Hello!';

Nil

Absence of a value

var empty = nil;

Object Literals

Object literals are collections of key-value pairs enclosed in curly braces ({}). Keys must be string literals, and values can be any valid expression. 

var person = {
  "name": "John Doe",
  "age": 30,
  "city": "New York"
};

Object literals can be used with the JSON module to serialize and deserialize data: 

use json;

var data = {
  "name": "John Doe",
  "age": 30,
  "city": "New York"
};

// Convert object to JSON string
var jsonString = json.stringify(data);

// Parse JSON string back to object
var parsed = json.parse(jsonString);

Operators

Neutron supports a variety of operators for arithmetic, comparison, and logical operations.

Arithmetic Operators

Basic Operations

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
var x = 10 + 5;  // 15
var y = 20 - 10; // 10
var z = 5 * 2;   // 10
var w = 10 / 2;  // 5

Comparison Operators

Equality & Comparison

  • == Equal
  • != Not equal
  • < Less than
  • <= Less than or equal
  • > Greater than
  • >= Greater than or equal
say(10 == 10); // true
say(10 != 5);  // true
say(5 < 10);   // true
say(15 >= 10); // true

Logical Operators

Boolean Logic

  • and Logical AND
  • or Logical OR
  • ! Logical NOT
say(true and false); // false
say(true or false);  // true
say(!true);          // false

var a = 5;
var b = 10;
say(a > 0 and b < 20); // true

Variables

Variables in Neutron are declared using the var keyword. They are dynamically typed and can hold any value. 

var name = "Neutron";
var version = 1.0;
var isReady = true;

// Variables can be reassigned
name = "Neutron Language";
version = 2.0;

Functions (Partially implemented)

Functions are defined using the fun keyword. Function definitions and calls are available, but return statements are currently under development.

Note: Function definitions and calls are available, but return statements are currently under development.

Defining Functions

fun greet(name) {
    say("Hello, " + name + "!");
}

Calling Functions

greet("Neutron"); // Prints "Hello, Neutron!"

Return Statement (Not yet implemented)

Note: Return statements are currently under development and not yet available in Neutron.

The return statement will be used to return a value from a function. 

fun add(a, b) {
    return a + b;
}

var result = add(5, 10);
say(result); // 15

Classes

Neutron supports object-oriented programming with classes. Classes can have properties and methods, and support the this keyword. 

class Person {
    var name;
    var age;

    fun setName(name) {
        this.name = name;
    }

    fun setAge(age) {
        this.age = age;
    }

    fun introduce() {
        say("Hi, I'm " + this.name + " and I'm " + this.age + " years old.");
    }
}

// Create an instance
var person = Person();
person.setName("Alice");
person.setAge(30);
person.introduce(); // Output: Hi, I'm Alice and I'm 30 years old.

Control Flow

Neutron provides several control flow statements for conditionals and loops.

If-Else Statements

The if statement executes a block of code if a condition is true. The else statement can be used to execute a block of code if the condition is false. 

if (x > 10) {
    say("x is greater than 10");
} else {
    say("x is not greater than 10");
}

While Loops

The while loop executes a block of code as long as a condition is true. 

var i = 0;
while (i < 5) {
    say(i);
    i = i + 1;
}

For Loops (Not yet implemented)

Note: For loops are currently under development and not yet available in Neutron.

The for loop is a more convenient way to write loops with a clear initializer, condition, and increment. 

for (var i = 0; i < 5; i = i + 1) {
    say(i);
}

Modules

Neutron supports modules, which allow you to organize your code into separate files. The use statement is used to import a module.

Using Modules

use math;

The math Module

The math module provides a set of mathematical functions. 

use math;

var sum = math.add(5, 3);        // 8
var diff = math.subtract(10, 4); // 6
var product = math.multiply(6, 7); // 42
var quotient = math.divide(15, 3); // 5
var power = math.pow(2, 3);       // 8
var root = math.sqrt(16);         // 4
var absolute = math.abs(-5);      // 5

// Trigonometric functions
var sine = math.sin(1.57);        // ~1
var cosine = math.cos(0);         // 1
var tangent = math.tan(0.785);    // ~1

// Logarithmic and exponential
var log = math.log(2.718);        // ~1
var exp = math.exp(1);            // ~2.718

// Rounding functions
var ceiling = math.ceil(3.2);     // 4
var floor = math.floor(3.8);      // 3
var rounded = math.round(3.6);    // 4

The sys Module

The sys module provides system-level functions for interacting with the operating system and environment.

Input/Output Functions

use sys;

// Read input without a prompt
var name = sys.input();

// Read input with a prompt
var age = sys.input("Enter your age: ");

File System Functions

use sys;

// File operations
sys.cp("source.txt", "destination.txt");  // Copy file
sys.mv("old.txt", "new.txt");             // Move/rename file
sys.rm("unwanted.txt");                   // Remove file

// Directory operations
sys.mkdir("new_directory");               // Create directory
sys.rmdir("old_directory");               // Remove directory

// File/directory checking
var exists = sys.exists("file.txt");      // Check if exists

// Read and write files
var content = sys.read("file.txt");       // Read file contents
sys.write("file.txt", "Hello World");     // Write to file
sys.append("file.txt", "More text");      // Append to file

System Information Functions

use sys;

var currentDir = sys.cwd();              // Get current directory
sys.chdir("/path/to/directory");         // Change directory
var envVar = sys.env("PATH");            // Get environment variable
var sysInfo = sys.info();                // Get system information

Process Control Functions

use sys;

sys.exit();                              // Exit with code 0
sys.exit(1);                             // Exit with code 1

// Execute system command
var result = sys.exec("ls -la");         // Returns output and exit code

Built-in Functions

Neutron provides several built-in functions that are available in the global scope:

say(value)

Prints a value to the console

say("Hello, World!");
say(42);
say(true);

int(string)

Converts a string to an integer

var num = int("42"); // num = 42
var zero = int("0");  // zero = 0

str(number)

Converts a number to a string

var text = str(42);   // text = "42"
var pi = str(3.14);   // pi = "3.14"

bin_to_int(string)

Converts a binary string to an integer

var num = bin_to_int("1010"); // num = 10
var eight = bin_to_int("1000"); // eight = 8

int_to_bin(number)

Converts an integer to a binary string

var binary = int_to_bin(10); // binary = "1010"
var eight = int_to_bin(8);    // eight = "1000"

char_to_int(string)

Converts a single-character string to its ASCII value

var ascii = char_to_int("A"); // ascii = 65
var space = char_to_int(" ");  // space = 32

int_to_char(number)

Converts a number (ASCII value) to a single-character string

var char = int_to_char(65); // char = "A"
var space = int_to_char(32); // space = " "

string_get_char_at(string, number)

Returns the character at the given index

var char = string_get_char_at("Neutron", 0); // char = "N"
var last = string_get_char_at("Hello", 4);    // last = "o"

string_length(string)

Returns the length of the string

var length = string_length("Neutron"); // length = 7
var empty = string_length("");          // empty = 0

Standard Library

Neutron comes with a comprehensive standard library that provides modules for common programming tasks.

Math Module

The math module provides mathematical functions and constants. 

use math;

// Mathematical functions
var result = math.sqrt(16);        // 4
var power = math.pow(2, 3);        // 8
var absolute = math.abs(-5);       // 5

// Trigonometric functions
var sine = math.sin(1.57);         // ~1
var cosine = math.cos(0);          // 1
var tangent = math.tan(0.785);     // ~1

// Logarithmic functions
var natural_log = math.log(2.718); // ~1
var exponential = math.exp(1);     // ~2.718

// Rounding functions
var ceiling = math.ceil(4.1);      // 5
var floor = math.floor(4.9);       // 4
var rounded = math.round(4.5);     // 5

The JSON module provides functions for parsing and generating JSON data. 

import "json";

// Parse JSON string to object
var jsonString = '{"name": "Neutron", "version": 1.0, "active": true}';
var data = json.parse(jsonString);

print(data.name);    // "Neutron"
print(data.version); // 1.0
print(data.active);  // true

// Convert object to JSON string
var obj = {"language": "Neutron", "year": 2024};
var jsonOutput = json.stringify(obj);
print(jsonOutput);   // {"language":"Neutron","year":2024}

HTTP Module

The HTTP module provides functions for making HTTP requests. 

import "http";

// Make a GET request
var response = http.get("https://api.example.com/data");
print(response.status);  // 200
print(response.body);    // Response content

// Make a POST request with data
var postData = {"key": "value"};
var postResponse = http.post("https://api.example.com/submit", postData);
print(postResponse.status);

Convert Module

The convert module provides data conversion utilities. 

import "convert";

// String conversions
var num = convert.toNumber("42");    // 42
var str = convert.toString(42);      // "42"

// Binary conversions
var binary = convert.toBinary(10);   // "1010"
var decimal = convert.fromBinary("1010"); // 10

// Character conversions
var ascii = convert.charToAscii("A"); // 65
var char = convert.asciiToChar(65);   // "A"

System Module

The sys module provides system-level operations and file system access. 

import "sys";

// User input
var name = sys.input("Enter your name: ");
var age = sys.input("Enter your age: ");

// File operations
sys.write("data.txt", "Hello, Neutron!");
var content = sys.read("data.txt");
print(content); // "Hello, Neutron!"

// File system operations
var exists = sys.exists("data.txt");  // true
sys.mkdir("new_directory");
sys.cp("data.txt", "backup.txt");

// System information
var currentDir = sys.cwd();
print("Current directory: " + currentDir);

var envVar = sys.env("PATH");
print("PATH: " + envVar);

// Execute system commands
var result = sys.exec("ls -la");
print(result.output);
print("Exit code: " + result.code);

Complete API Reference

This section provides a complete reference of all available functions, methods, and modules in Neutron.

Built-in Functions

print(value)

Outputs a value to the console followed by a newline.

Parameters: value (any) - The value to print

Returns: nil

print("Hello, World!");
print(42);
print(true);

len(value)

Returns the length of a string.

Parameters: value (string) - The string to measure

Returns: number - The length of the string

var length = len("Neutron"); // 7
var empty = len("");         // 0

type(value)

Returns the type of a value as a string.

Parameters: value (any) - The value to check

Returns: string - The type name

print(type(42));        // "number"
print(type("hello"));   // "string"
print(type(true));      // "boolean"
print(type(nil));       // "nil"
